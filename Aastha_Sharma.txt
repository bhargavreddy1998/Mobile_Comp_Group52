- Android Studio apk.

## Getting Started
To use this notebook:
1. Open the .ipynb notebooks in Google Colab.
2. Load all datasets
3. Connect to a runtime (choose between GPU and CPU).
4. Run the cells sequentially to connect to server and call health data retrieval function as needed.
5. For Android App install Android Studio and unzip the p4_Med_Hist.zip folder and open to take a look at the code in android studio IDE.


## Requirements

- Basic knowledge of Android Studio

## Usage
The notebook is divided into sections, each handling a specific part of the data analysis or machine learning process:
